# erfana voice and tone

The default house brand speaks the way the toolkit itself does: confident,
direct, and matter-of-fact. It is a neutral starting point you are meant to
replace with your own brand.

## Attributes

- **Confident** - state things plainly; no hedging.
- **Direct** - lead with the result, then the detail.
- **Matter-of-fact** - describe what something does, not how impressive it is.

## Avoid

- Hype and superlatives ("revolutionary", "world-class", "seamless").
- Filler ("it's worth noting that", "in order to", "at the end of the day").
- Title Case in running prose - use sentence case.

## Watermark

Motion and video outputs carry the canonical watermark `Created with erfana`.
This single literal lives in `brand.json` (`voice.watermark`); never hardcode a
different string in scripts.
