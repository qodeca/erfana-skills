# Support

Need help with erfana? Here is where to go.

## Getting started

- Read the [`README.md`](README.md) — install, the skill catalog, and typical workflows.
- Check the [troubleshooting section](README.md#troubleshooting) for common install / update issues.

## Questions and discussion

- **GitHub Discussions**: the best place for "how do I…", ideas, and show-and-tell.
- **GitHub Issues:** for reproducible bugs and concrete feature requests. Please search existing issues first, and use the issue templates.

## What does *not* go here

- **Security vulnerabilities** — do not open a public issue. Follow [`SECURITY.md`](SECURITY.md) (GitHub private vulnerability reporting).
- **Trademark / name-use questions** — see [`TRADEMARKS.md`](TRADEMARKS.md), then open a Discussion if still unsure.

## Response expectations

This is a community project maintained on a best-effort basis. There is no SLA. Clear, reproducible reports with steps and environment details get resolved fastest.
